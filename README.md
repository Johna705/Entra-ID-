# Entra-ID-
This lab demonstrates hands-on experience with Microsoft Entra ID (formerly Azure Active Directory). The project focuses on identity and access management tasks commonly performed by IT administrators, including user management, group administration, role assignments, and security configuration.

Skills Demonstrated

Microsoft Entra ID Administration
Identity and Access Management (IAM)
User and Group Management
Role-Based Access Control (RBAC)
Multi-Factor Authentication (MFA)
Microsoft 365 Administration
Security and Compliance Fundamentals
Troubleshooting User Access Issues

Lab Tasks Completed
Created and managed user accounts
Created security and Microsoft 365 groups
Assigned users to groups
Configured administrative roles
Enabled Multi-Factor Authentication (MFA)
Reset user passwords and managed account access
Reviewed sign-in logs and audit logs
Tested user access and permissions

Technologies Used
Microsoft Entra ID
Microsoft 365 Admin Center
Microsoft
Windows 11
Key Takeaways


This lab provided practical experience with cloud identity management and security administration. The project strengthened my understanding of how organizations manage users, permissions, authentication, and access to resources within Microsoft 365 environments.
